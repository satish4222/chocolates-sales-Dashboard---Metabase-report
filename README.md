Chocolate Sales Analytics Dashboard :
--This dashboard was created using Metabase to analyze and visualize chocolate sales data.

Features Included :
--Sales Trends: Revenue and volume trends over time
--Top Products: Best-selling chocolate items based on sales quantity and value
--Category Insights: Breakdown of product types or variants
--Seasonal Patterns: Purchase patterns by month/week
--KPIs: Core business metrics summarized for quick review

Purpose :
--The goal of this report is to transform raw sales data into meaningful insights that can support inventory
