#  Chocolate Sales Analysis Report (Metabase)

##  Overview

This project presents an interactive **Chocolate Sales Analysis Report** built using **Metabase**. The report provides clear insights into sales performance, shipment trends, and business metrics to support data-driven decision-making.

The dashboard is designed for **business users, analysts, and management** to easily explore sales data without writing SQL queries.

---

##  Tools & Technologies

* **Metabase** – Data visualization & dashboarding
* **MySQL** – Backend database
* **CSV Dataset** – Chocolate sales / shipments data
* **SQL** – Data cleaning, transformation, and analysis

---

##  Key Metrics Covered

* **Total Sales Revenue**
* **Total Shipments**
* **Sales by Product / Category**
* **Sales by Region / Country**
* **Monthly & Yearly Sales Trends**
* **Top Performing Products**
* **Low Performing Regions or Products**

---

##  Dashboard Features

* Interactive filters (Date, Region, Product)
* Time-series visualizations for trend analysis
* Bar charts and tables for product-wise comparison
* KPI cards for quick business insights
* Drill-down capability for detailed analysis

---

##  Data Source

* Data imported from **CSV files** into **MySQL**
* Dataset includes fields such as:

  * Order Date
  * Product Name
  * Category
  * Quantity
  * Sales Amount
  * Region / Country
  * Shipment Details

---

##  Data Preparation

* Removed null and inconsistent values
* Standardized date formats
* Cleaned product and region names
* Optimized tables for faster Metabase querying

---

##  Business Insights

* Identifies **peak sales periods**
* Highlights **high-revenue products**
* Helps detect **regional demand patterns**
* Supports inventory and supply chain decisions

4. Use visual insights for reporting and decision-making
